# Meet Me

Personal portfolio website built with Angular 21 and hosted on GitHub Pages.

The project showcases my passion projects, development style, and creative approach — from modern web applications to experimental game concepts and interactive experiences.

## Live Website

👉 https://uultimaa.github.io/meetMe

## Tech Stack

- Angular 21
- TypeScript
- SCSS
- Angular Signals
- GitHub Pages

## Features

- Fully static deployment (no backend required)
- Responsive modern UI
- Project showcase sections
- Interactive experience-focused design
- Fast GitHub Pages hosting

## Purpose

This portfolio was built to reflect both my technical skills and creative mindset.

Rather than functioning as a simple resume, it aims to present the way I think, design, and build experiences.
